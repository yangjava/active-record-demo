# Active Record Demo(活动记录)

## Active Record(活动记录)，是一种领域模型模式 </br>
   特点是一个模型类对应关系型数据库中的一个表，而模型类的一个实例对应表中的一行记录。</br>

## 面向领域对象的设计模式，为每个领域对象提供一组 CRUD 方法
